
# Dodo Payments Backend Assignment

Java Spring Boot + PostgreSQL implementation of:
- Invoice Service
- Mock PSP
- Webhooks
- Idempotent payments

## Run

```bash
docker compose up --build
```

## Services
- Invoice Service: localhost:8080
- Mock PSP: localhost:8081
- PostgreSQL: localhost:5432
